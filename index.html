hw-03: Interactive Webpage, and Marks & Channels

Marks

  • Areas: The states
  • Points: Nebraska's and Maine's electoral votes

Channels

  • Color
    • Represents attribute of quantitative type
    • We believe it's a good choice because it represents the data as intended, with the colors associating with their respective parties.
  • Size: Area
    • Represents attribute of quantitative type
    • We believe this is also a good choice because it shows the numerical difference in electoral votes, rather than simply which side a state chose.
  • Position: Both
    • Represents attribute of categorical type
    • We believe this isn't a bad choice, but it isn't super necessary. It shows where the votes are relative to each other.

Colormap(s)

  • Categorical
    • Represents the attribute of quantitative type: votes per state
    • Similar to what we wrote for Color, we believe this is a good choice because it connects the Color and Area channels.

Marks

  • Points: The data
  • Lines: Grouping of data

Channels

  • Position: Both
    • Represents the attribute of Ordered: Diverging type
    • We believe this is a good, and smart choice. It helps to show where the baseline of the data is, and the position each data point is relative to that base level.
  • Color: hue
    • Represents the attribute of Ordered: Diverging type
    • We believe this is bad choice because while purple and orange are two distinct colors, there seems to be no correlation between their selections. A scale from red to orange might have represented the data better.
  • Color: saturation
    • Represents the attribute of Ordered: Diverging type
    • We believe this is a good choice and should have been the original choice of representation for the Avg LWTS for a batted ball.

Colormap(s)

  • Divergent
    • Represents the attribute of diverging type: Avg LWTS for batted balls
    • Similar to what we wrote for Color: saturation, we believe this is a good choice because it effortlessly reperesents the groupings of data from each other, those being the below Avg, Avg, and above Avg LWTS.


Marks

  • Lines: Count of top rated movies from 2007-2022 by genre

Channels

  • Position: Both
    • Represents the attribute of categorical and quantitative type
    • We believe this is a good choice because it clearly separates the genres of movies and simply visualizes their magnitude.
  • Color: saturation
    • Represents the attribute of quantitative type
    • We believe this is a good choice because it helps to differentiate each movie genre from those with a drastic difference in count.

Colormap(s)

  • Sequential
    • Represents the attribute of quantitative type: count of top rated movies
    • Similar to what we wrote for Color: saturation, we believe this is a good choice because it easily distinguishes which genres are close to each other in count.
